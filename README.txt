INFM600This is the dataset created for the INFM600 Information Organization assignmentVersionVersion 1.0 (October 2015)DescriptionThis dataset is a derivative of the Capital_Project_Expenditures_and_Budgets__FY_2015 (https://opendata.howardcountymd.gov/Money-and-Finance/Capital-Project-Expenditures-and-Budgets-FY-2015/ghg6-4crr ), Capital_Project_Budget_Allocations__FY2016((https://opendata.howardcountymd.gov/Money-and-Finance/Capital-Project-Budget-Allocations-FY2016/5wgy-5dnq -) and Crime report_2015(((http://www.howardcountymd.gov/crimestats.htm). The datasets “Capital_Project_Expenditures_and_Budgets__FY_2015 “ and “Capital_Project_Budget_Allocations__FY2016” has the budget information of the years 2015 and 2016. The crime dataset contains all the information about the offenses that took place in the period of 2014 to June 2015  including the type of offenses. The original datasets are merged using category, project_title and budget_allocation. From the original dataset, only information necessary to analyze whether the increase or decrease in crime rate has effected the change in budget of 2016 has been maintained. This includes year, category, project_title, budget_allocations and total_offenses.  The dataset is released in the framework of INFM 600, Information Environments, Fall  2015, at the University of Maryland iSchool, http://ischool.umd.edu/mim. 


Research question
Does the crime incidents in the period of 2014 to June 2015 had an effect  in the planning of police department budget for the year 2016?  Files* Capital_Project_Expenditures_and_Budgets__FY_2015.csv     This file contains the category, year, project_title and the budget allocated for each project .                       * Capital_Project_Budget_Allocations__FY2016.csv     This file contains the category, year, project_title and the budget allocated for each project
* Semi-annual report_2015 .xlsx     This file contains the total number of crimes took place during the period of 2014 to June 2015.
* Budget_difference .xlsx   This file merges the two budget datasets of 2015 and 2016 and shows the difference.Data format* Capital_Project_Expenditures_and_Budgets__FY_2015.csv     category     count     capital_project_no    fiscal_year   project_title   spent/encumbered    budget                  * Capital_Project_Budget_Allocations__FY2016.csv     category     count     capital_project_no   project   revenue_score    budget_allocation  * Semi-annual report_2015.xlsx     2014     2015  
* Budget_difference.xlsx 	year	category	project_title	budget_allocationData statisticsThe proposed police budget has increased from $100,000.00 in 2015 to $2,000,000.00 in 2016. The total number of Part 1 crimes had decreased from 980.63 to 934.14 in the period 2014 to June 2015. The total number of Part 2 crimes had increased from 1520.3 to 1574.19 in the period 2014 to June 2015. Overall, the crimes had increased from 2500.9 to 2508.33 in the period 2014 to June 2015.Analysis:The analysis done on the datasets is represented graphically.
